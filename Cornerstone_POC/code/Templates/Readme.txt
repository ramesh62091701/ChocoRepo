To use "Template" and "APITemplate" as custom template in Visual Studio

Step-1 = Open the project
Step-2 = open the terminal
Step-3 = run the command "dotnet new --install . --force"
step-4 = open "Project" from menu bar and click on "Export Template"
step-5 = select "Project Template" option and click on Next
step-6 = Give Template Description and click on Finish
